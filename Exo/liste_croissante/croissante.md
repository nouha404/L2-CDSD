```
Exercice 
Écrire une fonction est_croissante qui détermine si une liste d’entiers passée en paramètre est croissante 
(i.e. renvoie True) et False sinon.

Qu’une liste L soit croissante signifie que L[i] ≤ L[i + 1] pour tout indice i pour lequel l’inégalitéa un sens.
``` 
```
Exemples :
[5, 6, 6, 6, 10] -> True [5, 6, 6, 6, 4, 5 ] -> False [42] -> True
```
